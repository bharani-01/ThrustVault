FAQs

Frequently Asked Questions

Common questions regarding the ThrustVault database console and security rules.

The console offers three active demo profiles for testing access levels: a Guest Profile (read-only), an Intern Profile (data editor), and a System Admin Profile (full privileges). You can quickly click the credentials cards on the login screen to autofill and sign in.

ThrustVault supports exporting motor lists in both CSV and JSON formats directly. For data imports, editors and admins can download a standardized Excel spreadsheet template, populate it, and upload it to add motors in bulk.

From the main specs grid, users can check boxes to select up to 3 motors. The system opens a bottom comparison drawer that displays a detailed side-by-side matrix comparing recommended ESCs, voltage configurations, propeller fits, and buy links.

Security is enforced at the database layer using PostgreSQL Row-Level Security (RLS) policies. While all users can read verified catalogs, write access to add, edit, or delete categories and motor specifications is strictly restricted to logged-in Interns and Admins.
