Import Requirements & Format

To ensure a successful bulk import, please make sure your spreadsheet matches the requirements below:

  • Required fields: A column for Motor Model Name is mandatory. If assigning categories By Row, a Category column is also required.
  • Extensions: Excel (.xlsx, .xls), CSV (.csv), or JSON formats are accepted.
  • Dynamic Columns: Any custom specs configured on ThrustVault (e.g. KV, weight) can be mapped dynamically from columns.
  • Duplicates: Select whether to skip or overwrite existing motors in the system using the strategy dropdown.
